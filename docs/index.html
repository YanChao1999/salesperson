What we provide

A hosted gateway between your website and LLM providers — not a library you import. Website owners get embed, API, and analytics in one platform.

Embeddable widget

Drop a script tag on Shopify, WordPress, or any static site. Visitors get a sales chat without you building frontend infrastructure.

Public chat API

POST /v1/chat/completions forwards messages through our gateway with tenant auth, behavior injection, and automatic token metering.

Sales behavior control

Configure system prompts, tone, and goals per website so each brand sounds on-message while sharing the same platform.

Usage & deal tracking

See messages, tokens, outcomes, and deal stages per site. Usage is recorded by the gateway — not self-reported by clients.

Multi-tenant by design

Each website gets its own API key, domain, LLM configuration, and isolated metrics — ready for a SaaS pricing model.

Provider flexibility

Route to OpenAI, Anthropic, or custom endpoints. Owners can bring their own keys or use platform-hosted inference later.

How it works

Website traffic never talks to the LLM directly. Everything goes through your platform API.

1

Register

Create a website tenant and receive an API key plus embed snippet.

2

Configure

Set sales prompts, tone, and goals that shape assistant responses.

3

Embed

Paste the script on your site. The widget calls our public API.

4

Measure

Track usage, outcomes, and deal progress from the summary API.

API surface

Two layers: a public product API for websites, and an admin API for setup.

Public API

  • POST /v1/chat/completions — forward chat (Bearer API key)
  • GET /v1/usage — usage summary for authenticated site
  • GET /widget.js — embeddable script

Admin API

  • POST /websites — register site, returns API key once
  • PUT /websites/{id}/behavior — sales prompt & goals
  • GET /websites/{id}/summary — dashboard metrics
Example chat request /v1/chat/completions
